Troubleshooting CSS Issues

Step 1: Clear Browser Cache

GitHub Pages heavily caches content. Try the following:

  • Press Ctrl+F5 (or Cmd+Shift+R on Mac) to force a hard refresh
  • Clear your browser's cache manually through browser settings
  • Try opening the site in an incognito/private window

Step 2: Check CSS Loading

Open your browser's Developer Tools (F12) and check:

  • Network tab: Is styles.css loading correctly?
  • Look for any 404 errors or failed resource loads
  • Check if the CSS file size matches what's expected

Step 3: Force CSS Update

You can force GitHub Pages to update your CSS by:

  • Adding a version query parameter to your CSS in the HTML
  • Example: <link rel="stylesheet" href="/css/styles.css?v=1">
  • Update the version number each time you make changes
